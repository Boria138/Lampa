# Changelog

Все заметные изменения в этом проекте фиксируются в этом файле.
Формат основан на [Keep a Changelog](https://keepachangelog.com/) и придерживается принципов [Semantic Versioning](https://semver.org/).

## [1.2.2] - 2026-01-27

### Security
- Исправлены уязвимости в зависимостях (lodash, tmp)
- Добавлена проверка origin для IPC-каналов `fs-existsSync` и `child-process-spawn` (доверяется только origin текущего `startUrl`)
- Усилен запуск внешних процессов: валидация имени команды и аргументов, безопасные `spawn`-опции без `shell`
- Ограничено открытие внешних ссылок и смена стартового URL только протоколами `http/https`

### Fixed
- Устранена утечка IPC listeners в `preload` для `child_process.spawn` (`stdout/stderr` отписываются при `exit/error`)

### Changed
- Обновлена версия Electron до v40.8.5
- Обновлена версия electron-store до v11
- Обновлена версия which до v6.1
- Проверка всех системных путей для отключения автообновления (не только /opt)
- Удалена архитектура из имени Mac релиза
- Обновлены GitHub Actions
- Добавлены Windows сборки в CI/CD
- Для Linux добавлена сборка `tar.gz` вместе с AppImage
